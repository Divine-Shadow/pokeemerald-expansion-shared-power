# Spoiler-Free Gameplay Changes

These notes cover gameplay rules and move access that are safe to know before starting a run.

## Move Changes

- Strength is now a Rock-type move.

## Move Access

- Yanmega now learns Hurricane at level 57.
- Metagross now learns Swords Dance at level 57 and from the Swords Dance tutor.

## Trainer Updates

- Gym leaders have been updated to better leverage Shared Power and more competitive movesets, while keeping their original teams and levels as intact as practical.
